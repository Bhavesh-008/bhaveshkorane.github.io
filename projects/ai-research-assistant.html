AI Research Assistant

An intelligent assistant combining real-time web search with AI-powered summarization and multi-turn conversation memory for contextual, dynamic information retrieval.

2025
Personal Project

Project Overview

The AI Research Assistant helps users get quick, reliable answers from the web without manually sifting through search results. Ask a question and the assistant searches the web in real-time, aggregates information from multiple sources, and provides a comprehensive, well-structured summary with citations.

The assistant maintains memory across interactions, enabling follow-up questions, deeper dives, and comparative analysis â€” all within a natural conversation flow. It leverages the Groq API for fast LLM inference and Serper API for web search, processed through LangChain.

Tech Stack

Python LangChain Streamlit Groq API Serper API

Key Features

Real-Time Web Search

Searches the web via Serper API and aggregates from multiple authoritative sources.

AI Summarization

Distills search results into concise, structured summaries with key insights.

Multi-Turn Memory

Remembers previous conversation context for natural follow-up questions.

Source Citations

Every response includes clickable source links for verification.

How It Works

1

User asks a research question

The user types their question in the Streamlit chat interface.

2

Web Search via Serper API

The system queries the web and retrieves the most relevant results.

3

LLM Summarization

Groq-powered LLM processes search results with conversation history to generate a comprehensive summary.

4

Response with Citations

Formatted response with source citations is presented, and memory is updated.
