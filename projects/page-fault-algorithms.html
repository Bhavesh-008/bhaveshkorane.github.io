Project Overview

This academic project implements and benchmarks three fundamental page replacement algorithms used in operating system memory management: FIFO (First-In First-Out), LRU (Least Recently Used), and OPT (Optimal). Understanding these algorithms is critical for OS design, as choosing the wrong strategy can drastically impact system performance.

The project accepts custom page reference strings and frame sizes as input, simulates each algorithm's behaviour step by step, counts page faults, and compares efficiency. A visual trace of each algorithm's decision-making makes it easy to understand how the page table evolves over time.

The comparative analysis revealed key insights about Belady's anomaly in FIFO and demonstrated why LRU is the most practical choice for real-world operating systems, performing close to the theoretical optimum.
