ML-Based College Prediction System

A machine learning system that predicts suitable colleges based on JEE scores using historical admission data â€” helping students make data-driven decisions about their higher education choices.

B.Tech Academic Project
Machine Learning

Project Overview

The ML-Based College Prediction System helps JEE aspirants make informed decisions during the college counselling process. By training on historical JEE admission data (cutoff ranks, branch preferences, and institutional records), the system predicts the likelihood of admission to specific colleges and branches for a given JEE rank.

Students input their JEE rank, category, and preferred branch, and the system returns a ranked list of colleges where admission is likely, possible, or difficult â€” giving them a realistic view of their options before the official counselling rounds begin.

The model was trained using multiple ML algorithms including Decision Tree, Random Forest, and KNN, with the Random Forest model performing best due to its ability to handle the complex, multi-factor nature of admission cutoffs. Feature engineering included normalising ranks by category and computing historical cutoff trends.

Tech Stack

Python scikit-learn Pandas NumPy Matplotlib Jupyter Notebook

Key Features

Multi-Model Comparison

Trained and compared Decision Tree, Random Forest, and KNN models to select the best-performing algorithm.

Category-Aware Predictions

Accounts for JEE reservation categories (General, OBC, SC, ST) with separate cutoff analysis for accuracy.

Historical Trend Analysis

Analyses multi-year admission cutoff trends to provide more reliable, trend-aware predictions.

Ranked College List

Returns a probability-ranked list of colleges categorised as "Safe", "Moderate", and "Reach" based on your rank.

How It Works

1

Data Collection & Preprocessing

Historical JEE admission data is cleaned, normalised, and feature-engineered (rank, category, branch, cutoffs).

2

Model Training

Multiple ML models are trained and evaluated; Random Forest selected for highest prediction accuracy.

3

User Input

User provides JEE rank, category, and preferred branch via the interface.

4

Prediction & Output

Model predicts admission probabilities for all colleges, returning a ranked list with confidence levels.
